Static site generator for Grunt.js and Yeoman. Assemble makes it dead simple to build modular sites, blogs, gh-pages, components and documentation from reusable templates and data.

Note that the YAML front matter is "borrowing" data from the `site` object, which is defined in the Grunt config. So in this example, `{{title}}` will yield the same result as `{{site.title}}`, and `{{description}}` will yield the same result as `{{site.description}}`
